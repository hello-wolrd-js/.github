# hello-world-js (你好世界)

🌍一个用于发现、探索、创造h5单页面应用的框架

🔨无开发框架限制, 可以支持使用任何的技术栈来开发

⚡使用pwa来提供接近原生app的使用体验

📦使用iframe模拟受控沙盒保障应用安全性

🎶提供丰富的API用于开发应用

✨内置 admin client server, 提供推荐部署方案
